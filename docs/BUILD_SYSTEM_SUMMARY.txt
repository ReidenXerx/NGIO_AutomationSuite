================================================================================
HOW TO BUILD
================================================================================

STEP 1: Install Dependencies (One Time)
----------------------------------------
pip install -r requirements.txt

This installs:
- PyInstaller (for creating .exe)
- psutil, colorama, colorlog (runtime)
- PyYAML, win10toast, tqdm (features)

STEP 2: Verify Build Readiness (Optional)
----------------------------------------
python test_build_readiness.py

This checks:
- Python version (3.8+)
- All dependencies installed
- Project structure intact
- PyInstaller working

STEP 3: Build Everything
----------------------------------------
build.bat

OR

python build_release.py

This creates:
- dist\NGIO_Automation_Suite_v1.5.0.exe (single file!)
- release\NGIO_Automation_Suite_v1.5.0_Release.zip (main package)
- release\NGIO_Automation_Suite_v1.5.0_Portable.zip (source)
- release\CHECKSUMS_SHA256.txt (integrity)

Build time: 2-5 minutes

STEP 4: Selective Builds (Optional)
----------------------------------------
python build_release.py --exe-only         # Only .exe
python build_release.py --portable-only    # Only source ZIP

================================================================================
TROUBLESHOOTING
================================================================================

Q: Build fails with "PyInstaller not found"
A: pip install pyinstaller

Q: Build fails with "Module not found"
A: pip install -r requirements.txt

Q: .exe size is huge (>100MB)
A: Use clean virtual environment (see BUILD_GUIDE.md)

Q: Build is slow
A: First build: 3-5 min (normal)
   Subsequent: 2-3 min
   Close other apps, use SSD

Q: .exe doesn't run
A: Check Windows SmartScreen - click "More info" → "Run anyway"
   Check antivirus - add exception if needed
